Get Started in Seconds

Highlighting your first code snippet takes less than 10 seconds

1

Install the Extension

Search for "Highlightability" in the VS Code Marketplace or click the install button above. One click and you're ready to go.

2

Select Code to Highlight

Open any file, select the code you want to mark, and press the toggle shortcut.

Ctrl+Shift+Z (Mac: Cmd+Shift+Z)
3

Add Comments (Optional)

Right-click any highlight to add notes and comments. Edit them anytime with the pencil icon.

Ctrl+Shift+C (Mac: Cmd+Shift+C)
4

Manage & Share

View all highlights in the sidebar. Export to share with your team, or import highlights from teammates.
